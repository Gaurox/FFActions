Fast right-click media tools for Windows

FFActions adds simple context-menu actions to process video, audio and image files without opening heavy software.

Download View Source

Right-click. Choose an action. Done.

FFActions integrates directly into the Windows context menu. Each operation creates a new file next to the original one and never overwrites your source file.

Features

Video

Cut by frame, cut by time, interpolate, remove audio, extract audio, rotate, flip, compress and convert.

Audio

Trim, convert, change speed, change pitch, reverse and compress audio files.

Image

Resize, crop, convert, compress, rotate, flip and generate icon files from images.

Built for quick everyday tasks

FFActions is not a full video editor, audio workstation or image editor. It is designed for fast operations with short dialogs, focused options and simple output handling.
